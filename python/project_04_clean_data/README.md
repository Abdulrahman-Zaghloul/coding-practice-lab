# Python Exercise 04 - Clean Data

## Objective

Practice filtering records and handling missing values.

## Problem

Given a list of records, return only those with a valid age value.

## Concepts Practiced

- Functions
- Loops
- Conditional statements
- Working with dictionaries
- Handling `None` values

## Key Takeaway

Data Engineers frequently clean datasets by removing or filtering records with missing or invalid data before loading them into downstream systems.
