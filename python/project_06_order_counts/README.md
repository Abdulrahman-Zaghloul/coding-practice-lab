# Python Exercise 06 - Count Orders by Status

## Objective

Practice counting records by category using a dictionary.

## Problem

Given a list of orders, count how many orders exist for each status.

## Concepts Practiced

- Functions
- Loops
- Dictionaries
- Counting occurrences
- Aggregation logic

## Key Takeaway

Dictionaries are useful for grouping and counting records, which is a common pattern in data processing and ETL pipelines.
