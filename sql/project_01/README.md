# SQL Exercise 01 - Sales Basics

## Objective

Practice using:

* SELECT
* WHERE
* ORDER BY

## Problem

Query sales records where the amount is greater than 100 and sort the results from highest to lowest amount.

## Concepts Practiced

* Filtering data
* Selecting specific columns
* Sorting query results

## Key Takeaway

The combination of `SELECT`, `WHERE`, and `ORDER BY` forms the foundation of most SQL queries and is commonly used in analytics and data engineering workflows.
